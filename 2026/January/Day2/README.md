# Day 02 – Array Basics & Two Pointers

## Problems Solved
1. Largest Element in an Array
2. Second Largest Element in an Array (without sorting)
3. Check if the Array is Sorted
4. Remove Duplicates from Sorted Array (LC 26)
5. Move Zeros to End (LC 283)

## Patterns / Concepts
- Array traversal
- Single-pass tracking
- Two pointers (in-place)
- Monotonic property

## Key Learnings
- Tracking multiple values (largest & second largest) can be done in one pass
- Sorted property simplifies many problems
- Two pointers avoid extra space and improve efficiency
